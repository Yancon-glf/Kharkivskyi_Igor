# Портфоліо (GitHub Pages)

## Як опублікувати
1. Створи репозиторій на GitHub (звичайний, наприклад: `portfolio-ihor`).
2. Завантаж у репозиторій **весь вміст цієї папки** (index.html, resume.html, report.html, assets/).
3. Відкрий: **Settings → Pages**
4. Source: **Deploy from a branch**
5. Branch: **main** / Folder: **/(root)**
6. Збережи — GitHub покаже посилання на сайт.

## Структура
- `index.html` — перша сторінка (інформація про студента)
- `resume.html` — резюме
- `report.html` — аналітичний звіт (5 загроз + 5 рекомендацій)
- `assets/style.css` — стилі
- `assets/main.js` — підсвічування активного пункту меню
